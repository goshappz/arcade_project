# Слизни против Фруктов

Игра в жанре **Tower Defense** на Python с использованием библиотеки Arcade.  
Фрукты защищают свою родную рощу от слизней и улиток, которые решили полакомиться сочными фруктами.

---

## Сюжет

Слизни давно присматривались к свежим яблокам, грушам, вишням и другим фруктам. 
Сначала они тихо подползали и пытались стянуть пару фруктов, далее все переросло в откровенное воровство, так и началась эскалация конфликта.

Оскорблённые жадностью фруктов, слизни начали готовить полноценное **вторжение** в рощу.  
В ответ фрукты объявили **мобилизацию**: боевые деревья заняли позиции вокруг рощи.  
Последняя линия обороны — **деревянный забор**, в который теперь долбятся орды врагов.

Если забор падёт — роща потеряна.  
Твоя задача — **спасти фрукты, товарищ командир**.

---

## Геймплей

- Враги идут по пути к **деревянному забору**.
- Ты размещаешь **фруктовые башни**, которые автоматически атакуют врагов.
- За уничтоженных врагов падают **деньги**:
  - Используются для постройки новых деревьев и улучшения обороны.
- Если здоровье забора падает до нуля — **поражение**.

---

## Башни

Каждая башня — это дерево со своим типом атаки. Пока незавершенно

- **Яблоня**  
  Базовое боевое дерево.  
  - Стреляет **одиночными яблоками** по одному ближайшему врагу.
  - Дёшево стоит, основа любой обороны.

- **Грушевое дерево**  
  Использует **отскакивающие груши**.  
  - Груша после попадания может отскочить и задеть другого врага.
  - Эффективно против плотных скоплений противников.

- **Вишня**  
  Оружие массового поражения.  
  - Выпускает **залп вишен по области**.
  - Наносит урон сразу нескольким врагам в радиусе.

---

## Враги
Пока незавершенно
- **Слизень**  
  - Стандартный враг.
  - Средняя скорость и здоровье.
  - Основная масса атакующих.

- **Улитка**  
  - Медленная, но **бронированная**.
  - Получает меньше урона от атак.
