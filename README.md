# arcade_project
Проект Яндекс Лицей Arcade
Участники:
  Попов Георгий (Тимлид)
  Ушаков Иван  
Тема: Tower Defense  
Стратегическая игра, цель - не дать врагам дойти и сломать главное здание, раставляя и улучшая башни. Враги становятся сильнее со временем.  
С врагов падает валюта, за которую покупаются башни и улучшения.  
Сущеествуют разные виды башен и врагов. Разные виды башен эффективно справляются с определенными врагами, а другие наоборот хуже.
